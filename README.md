# DKIP.PM.02.MDK.01-TRPOLab
Учебный репозиторий с методическими рекомендациями по производственной практике **ПМ.03 Проектирование, разработка и оптимизация веб-приложений **  для студентов СПО по специальности 09.02.07 «Информационные системы и программирование» группы ДКИП-401ГДРМИ


Репозиторий используется для:
- Хранения данных по лабораторным работам
- Описанию итоговых отчетов для сдачи
